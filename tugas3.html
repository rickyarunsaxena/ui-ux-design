Find 10 more attribute and how to use it

Atribut HTML

Atribut Pilihan Fungsinya
 align  right, left, center  Rata secara horizontal
 valign  top, middle, bottom  Rata secara vertikal
 bgcolor  numeric, hexidecimal, RGB values  Memberi warna latar elemen
 background  URL  Memberikan background gambar elemen
 id  Diatur pengguna  Nama khusus sebuah elemen untuk CSS
 class  Diatur pengguna  Nama untuk pengklasifikasian CSS
 width  Nilai numeric  Biasanya untuk lebar table dan img
 height  Nilai numeric  Biasanya digunakan untuk tinggi table dan img.
 title  Diatur pengguna  Judul dari elemen "Pop-up"
 src  Diatur pengguna  Atribut untuk menentukan URL gambar
 ltr  Left to Right  Kiri ke kanan (nilai default)
 rtl  Right to Left  Kanan ke kiri (untuk bahasa seperti Ibrani ata Arab yang dibaca dari kanan ke kiri)
 alt  Diatur Pengguna  Menentukan teks alternatif untuk gambar
 disabled  Diatur Pengguna  Menentukan bahwa elemen masukan harus dinonaktifkan
 href  URL  Menentukan URL (web address) untuk link
